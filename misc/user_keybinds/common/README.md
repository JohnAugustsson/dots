# common

Reference copies of the remap definitions.

## Files

- `symbols/se-ja-keyd-like` — X11-style symbols file
- `ja-se-keyd-like.full.xkb` — full keymap file for wlroots-style `xkb_file` / `kb_file` use

These are not applied automatically.
Use the environment-specific directories for actual per-session instructions.
